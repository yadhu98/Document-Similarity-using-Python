A software process is sequence of actions that lead to the production of a software. The importance of software process is that it helps in deploying a project with user satisfactions and within time of the project. The various software processes that are followed are communication that is finding out the need of  the users. Planning, planning the software. Modelling building a generic model of the software. Construction,constructiing the software. Deployment, deploying software. It is important because inorder to complete product in time and as required by the client we need a well defined process which can make a software error free and fast.
